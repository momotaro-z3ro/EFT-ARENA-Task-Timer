## 🌟 Botの紹介

**EFT & ARENA Task Timer Bot** 

毎日のタルコフライフで、「今日のデイリータスクあと何時間猶予あるっけ？」を解決するために作成しました。

あなたが **Escape from Tarkov** や **Tarkov: ARENA** を起動するだけで、Discord上のステータスを自動検知！
バックグラウンドでカウントダウンタイマーをスタートさせ、時間が近づくとDMでリマインダー通知をお届けします。

さらに、現在受諾しているタスクの「内容」まで書き込んで管理できます！

---

## ✨ 主な機能

- 🕹️ **ゲーム起動の自動検知**:
  TarkovやARENAをプレイし始めると、自動的に24時間（または1週間）のタスクタイマーの監視が開始されます。

- 📝 **個別タスクのスケジュール管理**:
  `/eft about_task` などの専用コマンドで、現在受けている個別のタスク内容をBotに記録できます（デイリー最大4つまで対応など）。

- ⏰ **柔軟なリマインダー通知**:
  「終了の3時間前」や「10分前」など、秒単位で細かく通知時間を設定可能。「毎回」鳴らすか、「今回だけ」鳴らすかも自由に選べます。

- 📊 **ステータス確認**:
  `/eft status` や `/arena status` で、現在の残り時間と完了したタスクの進捗状況をいつでもひと目で確認可能！

## ⚠️ 利用前の準備（必読）

本Botは、Discord機能の「プレイ中のゲーム（Rich Presence）」ステータスを読み取って自動動作する仕組みになっています。
そのため、ご利用になる際は**必ずDiscordの「アクティビティステータス」を公開設定（オン）**にしてください。

### ⚙️ 設定手順
1. Discord左下の **歯車マーク（ユーザー設定）** を開く
2. 左メニューの下の方にある **「アクティビティのプライバシー」** をクリック
3. **「現在のアクティビティをステータス メッセージに表示する」をオン** にする

> 🚫 **注意**: これがオフになっていると、あなたがゲームを起動してもBot側から検知できず、自動タイマーが開始されません！
